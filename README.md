# Expense Tracker
Projekt na pracownię języków skryptowych
Grupa 1


## Tools:
 - Makefile:   
  `make frontend` -> builds and starts fronted
  `make backend` -> builds and starts backend
 - Access to Prod database:
 1. Install Render CLI: `curl -fsSL https://raw.githubusercontent.com/render-oss/cli/refs/heads/main/bin/install.sh | sh`
 2. Access: `render psql dpg-<omitted>-a`
